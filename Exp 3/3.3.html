Introduction to JavaScript

JavaScript is a programming language used to create dynamic web pages.

It runs in the browser and allows interaction with users.

console.log("Hello, world!");
  • Client-side language
  • Interpreted
Variables and Data Types

Variables store data values.

JavaScript supports multiple data types.

let age = 25;
  • String
  • Number
Functions

Functions allow code reuse.

They can accept parameters.

function greet() { return "Hi"; }
  • Reusable blocks
Loops

Loops repeat code execution.

They reduce repetition.

for (let i = 0; i < 5; i++) {}
  • for loop
Arrays

Arrays store multiple values.

They are zero-indexed.

let colors = ["red", "blue"];
  • Indexed collection
